# Titanic Survival Prediction — Streamlit ML Demo https://sbjbfvkmdlwmdfweb2cqwp.streamlit.app/


Это интерактивное веб-приложение, построенное на **Streamlit**, которое позволяет пользователю ввести данные пассажира Титаника и получить предсказание — выжил бы он или нет — на основе обученной модели машинного обучения.

---

## Цель проекта

Создать удобный и визуально привлекательный интерфейс, где пользователь может:

- Ввести свои гипотетические данные (пол, возраст, класс каюты и т.д.)
- Получить предсказание выживания
- Увидеть вероятность выживания в процентах
- Ознакомиться с аналитикой по реальным данным пассажиров Титаника

---

## Модель машинного обучения

### Архитектура модели

Используется **Stacking Classifier** — ансамблевая модель, объединяющая:

- **XGBoost Classifier** — мощный градиентный бустинг
- **Random Forest Classifier** — ансамбль деревьев решений
- **Logistic Regression** — в качестве мета-классификатора, объединяющего предсказания базовых моделей

### Оптимизация гиперпараметров

Для поиска лучших гиперпараметров используется **RandomizedSearchCV** с 20 итерациями и 3-кратной кросс-валидацией. Оптимизируются параметры:

- `learning_rate`, `n_estimators`, `max_depth` для XGBoost
- `n_estimators`, `max_depth` для Random Forest
- `C` (регуляризация) для Logistic Regression

### Подготовка данных

- Удалены столбцы: `Name`, `Ticket`, `Cabin`, `Fare` (неинформативные или с большим количеством пропусков)
- Пропуски в `Age` заполнены медианой, в `Embarked` — модой
- Категориальные признаки (`Sex`, `Embarked`, `Pclass`) закодированы через `pd.get_dummies()`
- Признаки масштабированы с помощью `StandardScaler`

### Точность модели

Модель достигает точности **~80-82%** на валидационной выборке (значение выводится в интерфейсе приложения).

## Интерфейс приложения

Приложение разделено на две части:

### 1. Аналитические графики

Отображаются 4 графика, визуализирующие влияние различных факторов на выживаемость:

- **Распределение возраста** по выжившим/погибшим (с KDE)
- **Выживаемость по полу**
- **Выживаемость по классу каюты**
- **Выживаемость по порту посадки**

> Графики построены с помощью `seaborn` и `matplotlib`, стилизованы и снабжены пояснениями.

### 2. Интерактивный предиктор

В боковой панели (`sidebar`) пользователь может задать:

- Пол (`Мужской` / `Женский`)
- Класс каюты (`Первый`, `Второй`, `Третий`)
- Возраст (от 1 до 80 лет)
- Порт посадки (`Шербур`, `Квинстаун`, `Саутгемптон`)
- Количество братьев/сестёр/супругов на борту
- Количество детей/родителей на борту

После ввода данных:

- Формируются признаки, аналогичные обучающей выборке
- Применяется масштабирование
- Делается предсказание и выводится:

  - Текстовый результат: *«Сожалеем, вам не повезло»* или *«Ура! Вы будете жить»*
  - Вероятность выживания в виде прогресс-бара и процентов
