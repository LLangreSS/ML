# Категоризация альбомов по кумулятивной доле продаж (ABC-анализ)

---

## 📌 Суть метода

ABC-анализ основан на **принципе Парето (80/20)** и позволяет ранжировать объекты по их вкладу в общий результат.  
В контексте музыкальных альбомов — это классификация по объёму продаж:

- **Категория A**: Топовые альбомы, формирующие **~80%** от общих продаж.  
- **Категория B**: Средние по вкладу — в сумме дают **~15%**.  
- **Категория C**: Остальные альбомы, вносящие лишь **~5%** в общий объём.

Такой подход помогает сфокусировать ресурсы на наиболее значимых позициях.

## 🔧 Алгоритм реализации

1. **Сортировка** альбомов по убыванию продаж (`ORDER BY sales DESC`).  
2. **Расчёт кумулятивной доли** с помощью оконной функции:  
   ```sql
   SUM(percentage) OVER (ORDER BY sales DESC ROWS UNBOUNDED PRECEDING)```
3. **Присваиваение** категории каждому альбому в зависимости от значения кумулятивной суммы.

## 🚀 Практическое применение

1. **Приоритизация маркетинговых усилий** — фокус на альбомах категории **A**, приносящих основной доход.  
2. **Оптимизация управления складскими запасами** — поддержание достаточного уровня популярных релизов.  
3. **Планирование новых релизов и промо-кампаний** — анализ прошлых успехов для стратегического принятия решений.
